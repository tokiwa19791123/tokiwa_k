ツールチップ

ツールチップは、テキストにマウスホバーすることで現れる説明文です。 上に出てくる、右に出てくる、下に出てくる、左に出てくる、というようにホームページの内容やデザインに合わせて、自由にカスタマイズできます。 
._tooltip {
  display: inline-block;
  position: relative;
  color: #06c;
  text-decoration: underline;
}
._tooltip::before {
  display: none;
  position: absolute;
  content: attr(data-tooltip);
  padding: 5px;
  line-height: 1.0;
  border-radius: 5px;
  background: #fc3;
  color: #222;
  font-size: 0.8em;
  z-index: 10;
}
._tooltip::after {
  display: none;
  position: absolute;
  content: "";
  border: 5px solid transparent;
  z-index: 10;
}
._tooltip:hover::before, ._tooltip:hover::after {
  display: block;
}
._tooltip._tooltip_top::before {
  top: -2.0em;
}
._tooltip._tooltip_top::after {
  top: -3px;
  left: calc(50% - 10px);
  border-top: 5px solid #fc3;
}
._tooltip._tooltip_right::before {
  right: -100%;
}
._tooltip._tooltip_right::after {
  top: calc(50% - 5px);
  right: -10px;
  border-right: 5px solid #fc3;
}
._tooltip._tooltip_bottom::before {
  bottom: -2.0em;
}
._tooltip._tooltip_bottom::after {
  bottom: -3px;
  left: calc(50% - 10px);
  border-bottom: 5px solid #fc3;
}
._tooltip._tooltip_left::before {
  left: -100%;
}
._tooltip._tooltip_left::after {
  top: calc(50% - 5px);
  left: -10px;
  border-left: 5px solid #fc3;
}
